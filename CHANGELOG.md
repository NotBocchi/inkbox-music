# Changelog

## INKBOX Music v1.1.2

### 新增与优化

- 新增 LRCLIB 在线歌词搜索、绑定与本地缓存，保留本地歌词优先。
- 正在播放页新增在线获取歌词入口，减少从三点菜单查找的成本。
- 简化三点菜单和封面系统：保留内嵌封面优先，无封面时使用全局默认渐变封面。
- 设置页新增默认渐变封面预设与歌词模糊开关。
- 播放栏进度条左侧新增“喜欢”和“添加到歌单”快捷按钮。

### 修复

- 修复安装版在线获取歌词会立即失败的问题。
- 修复桌面歌词窗口透明圆角的白色边角问题。

### 说明

- 跳过 v1.1.1：该版本测试中 bug 较多、体验较差，未作为正式发布版本。
